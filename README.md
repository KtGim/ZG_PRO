# 个人笔记网站

## 项目目录

- builders 构建工具配置目录
- components 组件库
- config 配置文件
- consts 常量文件
- dist 产物
- docs 文档
- loaders 自定义 loader
- plugins 自定义 plugin
- logs 日志
- site 文档网站项目
